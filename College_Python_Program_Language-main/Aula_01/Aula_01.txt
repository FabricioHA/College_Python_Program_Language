				Paradigimas

Siginifica modelos, padrão.

Toda linguagem estão dentro e um paradigima, um padrão, como orientação a 
objeto, funcinal,imperativa, lógica, etc.

-Programação imperativa: Um tipo de programação sequencial, como ações, enunciados
ou comandos que mudam o estado (variaveis) de um programa.

-Programação orientada a objeto: Seus principais pilares são abstração, 
encapsulamento, herança, poliformismo

-Programação funcional: É um tipo de programação que se concentra na avaliação de 
expressões e naaplicações de funções matemáticas.

Programação lógica: É um tipo de linguagem utilizada para desenvolver intruções em sequencia para atingir determinado objetivo.

==================================================================================

				Executável

É uma compilação do algoritomo escrito, com o intuito de realizar uma tarefa específica, sendo seu caso exibir as funções e instruções do programa.

Para gerar um executável, normalmente utilizamos compiladores, que traduz um
programa descrito em uma linguagem de alto nível para um executável.
